# Hello World: Building My Portfolio

Welcome to my first blog post! In this series, I'll be documenting the process of building my AI-powered, Next.js portfolio.

## The Architecture
We are using some cutting-edge tools:
- **Next.js 15+ (App Router)**
- **Tailwind CSS v4**
- **Framer Motion**
- **Supabase** for Auth & Database
- **Google AI Studio** for an intelligent assistant

### Why Next.js?
Next.js provides an incredible developer experience and out-of-the-box performance optimizations like Server Components and advanced caching.
