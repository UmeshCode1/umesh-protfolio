# Umesh Patel Portfolio

A high-performance, anime-inspired, minimal, intelligent portfolio platform.

## Features
- Dynamic content system
- Blog (MDX-based)
- GitHub activity integration
- Admin dashboard (CRUD)
- AI assistant (context-aware)
- Clean GitHub automation

## Tech Stack
- Frontend: Next.js (App Router)
- Styling: Tailwind CSS
- Animation: Framer Motion
- Backend: Appwrite (DB + Storage)
- Auth: Clerk
- AI: Google AI Studio
- Deployment: Vercel

## Local Development
1. Clone the repository
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env.local` and fill in the values
4. Run the development server: `npm run dev`

## Commit Format
Please follow this commit format: `type(scope): message`
Examples:
- `feat(ui): add hero section`
- `fix(api): resolve data fetching bug`
- `chore: update dependencies`
